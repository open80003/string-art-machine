## 🚀 طريقة النشر على GitHub Pages

### الخطوة 1: إنشاء مستودع على GitHub
1. اذهب إلى [GitHub](https://github.com)
2. اضغط على زر "New repository"
3. اختر اسم للمستودع مثلاً: `string-art-machine`
4. اضغط "Create repository"

### الخطوة 2: رفع الملفات
```bash
# انسخ هذه الأوامر في terminal
git clone https://github.com/اسم-المستخدم/string-art-machine.git
cd string-art-machine
# انسخ كل ملفات المشروع هنا
git add .
git commit -m "إضافة موقع التحكم"
git push
