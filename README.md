# 🎨 نظام التحكم بآلة String Art

موقع تحكم كامل لآلة String Art يعمل على GitHub Pages ويتواصل مع ESP32 عبر MQTT.

## ✨ المميزات

- ✅ واجهة عربية كاملة
- ✅ تحكم بالرسم والتثقيب
- ✅ رفع ملفات الرسم
- ✅ مراقبة لحظية للحالة
- ✅ تحكم يدوي كامل
- ✅ إعدادات قابلة للتعديل

## 🚀 طريقة النشر على GitHub Pages

1. ارفع الملفات الستة إلى مستودع GitHub
2. اذهب إلى Settings → Pages
3. اختر main branch و /root
4. اضغط Save

رابط موقعك: `https://اسم-المستخدم.github.io/string-art-machine`

## ⚙️ الإعدادات المطلوبة

في ملف `config.js`، تأكد من:
- `MACHINE_ID` يطابق المعرف في كود ESP32
- `USE_EMQX: true` مفعل

## 📞 الدعم الفني

للاستفسارات والمشاكل، يرجى فتح Issue في GitHub.
